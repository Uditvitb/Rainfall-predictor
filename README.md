Rainfall Prediction System

Overview

This is a project which predicts rain or not on the basis of the weather variables such as weather temperature, humidity, pressure and the likes through machine learning.
Features
Takes real-time user input
Identifies with the help of decision tree classifier.
Output simple: will/ will not Rain
Tech Stack
Python
pandas
scikit-learn
Project Structure
main.py        # User input & output
score.py       # Results and predictions of the solvent model.
Rainfall.csv   # Dataset
Setup
pip install scikit-learn pandas.

Run
python main.py

Working
Data is pre-processed and cleansed.
Decision Tree use in training the model.
Model is fed with user input upon which prediction is carried out.
Output
1 - will Rain
0 - will not rain
